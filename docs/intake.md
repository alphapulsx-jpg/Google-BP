# Intake form (post-payment)

## Tool
- **Tally / Google Forms** — link only **after** Stripe payment (redirect URL).

## Required fields
1. Business legal name + DBA
2. **Google Maps / GBP link** (or "none — need help claiming")
3. Service address(es) + **service area** (zip/cities)
4. Top **10 services** (exact phrases customers use)
5. Primary phone + monitoring email
6. Hours + holiday exceptions
7. **GBP access:** invite **this email:** **[your GBP-manager email]** as **Manager** (instructions link: Google Help "add managers").
8. Competitor names (up to 3) — optional
9. Brand voice: **Professional / Friendly / Premium / Budget** (pick one)
10. Photo permission: yes/no + link to Drive folder if uploading

## Stop / disqualify (auto-reply template)
- Cannot grant access within 7 days → offer refund per policy.
- Wrong geography → refund.
- Wrong industry (outside ICP) → refund.

## Auto-response email (short)
- Thanks + **what happens next** + **expected delivery date** + link to upload photos.
