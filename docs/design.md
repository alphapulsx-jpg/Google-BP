# Design: GBP DIY kit (brochure + report)



## North star

- **Revenue goal:** adjust targets to **~$390 in first 30 days** → roughly **3× $129.99** upfront (example only).

- **Time budget:** ≤ **[hours/week]**; ops batched in short daily checks (not a “weekend project” for customers).

- **Interaction model:** **Email only** — no sales calls; **forms + payment link**.



## Offer (fixed scope)

- **Name:** $129.99 Google Business Profile DIY kit — **printable brochure + written report**, **delivered in ~90 seconds** (after free scan + pay; automated AI scan)

- **ICP:** **[Trade]** owners in **[City/region]**, **small to medium businesses under 100 employees**, already serving customers, **weak GBP** (thin profile) OR strong offline signals but **missing/duplicated GBP** (handled cautiously in scoring).

- **Included:** Prioritized checklist, paste-ready headline/services/description snippets, categories/services guidance (they apply), attributes where applicable, photo shot list, Q&A prompts, **5 post drafts**, light competitor positioning notes, **visual brochure** (print/PDF/export) paired with step-by-step copy blocks in **one Google Doc** from template — **customer applies all changes**.

- **Excluded:** logging into client GBP, requesting manager invitations, hands-on edits, website rebuild, ads, SEO content machine, “#1 ranking” guarantees, duplicate suppression legal claims, phone support.



## Pricing & policy

- **Price:** $129.99 **upfront** (Stripe Payment Link; amount must match the link).

- **Refund:** Full refund only if brochure + written report are **not delivered within 5 business days** of completed post-pay form (policy also on landing page).



## SLA

- **Customer-facing promise:** **~90 seconds** from successful payment + valid listing identifier → brochure/export + report by email (automated scan).

- **Link guarantee:** if the Maps listing link cannot be read, customer is **not charged** (automatic refund).

- **Customer time upfront:** **~10 seconds** for free scan (paste link or name + city); copy-paste steps are on their schedule after delivery.

- **Internal ops:** Kits are **agent-generated** from listing snapshot; **optional human spot-check** before send if approval gate enabled; automation target **~90s** after valid payment.

- **Your responses:** **[e.g. Tue/Thu 6–7pm + Sun]** email only; no calls.



## Tools (no custom app)

- Carrd or Google Sites (landing)

- Stripe Payment Link

- Tally or Google Forms (intake)

- Google Docs + Drive / export to PDF for brochure artifact

- Google Sheet (pipeline)

- Google Places API **or** permitted export + manual review (see `lead-score.md`)

- Agent workflow for report/brochure draft (human QA optional)



## Metrics

- Landing: visits → payment

- Ops: scan → paid → valid listing → report delivered (~90s automated)

- Growth: leads/week, message reply rate, **pay rate**



## Agent / builder rules

- Do **not** expand scope beyond this doc.

- Any new automation must **reduce hours** or **increase payment conversion** — otherwise defer.
